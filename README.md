# SafeGuard Senior — On-Device Fraud Protection

🛡️ **SafeGuard Senior** is a privacy-first Android app that protects users against SMS and browser fraud using local AI (Odin-Ai) and official Belgian cybersecurity sources.

**Status:** Beta testing — we're looking for daily testers to provide feedback.

---

## Quick Start

### Download
- **Google Play:** [Download SafeGuard Senior](https://play.google.com/store/apps/details?id=com.safeguard.senior)

### Website
This is the landing page for SafeGuard Senior.

**Local development:**
1. Clone or download this repository
2. Open `index.html` in your browser
3. No build step required — just static files

---

## Key Features

✅ **100% On-Device** — No cloud, no account, no tracking  
✅ **Odin-AI** — Local threat detection that learns from daily fraud reports  
✅ **SMS & Browser Protection** — Real-time safeguards against malicious links  
✅ **Battery-Aware** — Optimized animations for low-battery scenarios  
✅ **Official Belgian Sources** — Updates from Meldpunt, CCB, Safeonweb  
✅ **Multi-Device** — Optimized for Xiaomi, OnePlus, Oppo, Vivo, Huawei, Samsung, Pixel  

---

## Official Sources

SafeGuard Senior pulls threat data from:

- **[Meldpunt.belgie.be](https://www.meldpunt.belgie.be/)** — Federal fraud reporting
- **[Centre for Cybersecurity Belgium (CCB)](https://www.safeonweb.be/)** — Cyber awareness & warnings
- **[Safeonweb](https://www.safeonweb.be/)** — Official Belgian cybersecurity guidance
- **Report suspicious:** [suspicious@safeonweb.be](mailto:suspicious@safeonweb.be)

## Customization

### Colors & Branding
Edit `assets/styles.css` — CSS variables at the top:

```css
:root {
  --primary: #6366f1;          /* Main color */
  --secondary: #10b981;         /* CTA button */
  --danger: #ef4444;            /* Alerts */
  /* ... more variables */
}
```

### Dark Mode
Automatic based on system preference. Users can toggle via the 🌙 button (stored in localStorage).

### Content
Edit `index.html` to change:
- Hero text & CTAs
- Feature descriptions
- About section
- Social links
